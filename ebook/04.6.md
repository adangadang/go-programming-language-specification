# 4.6 操作符/运算符和分隔符

下面的一些字符序列被当做操作符/运算符、分隔符或是其他一些特殊的符号：

	+    &     +=    &=     &&    ==    !=    (    )
	-    |     -=    |=     ||    <     <=    [    ]
	*    ^     *=    ^=     <-    >     >=    {    }
	/    <<    /=    <<=    ++    =     :=    ,    ;
	%    >>    %=    >>=    --    !     ...   .    :
	     &^          &^=
